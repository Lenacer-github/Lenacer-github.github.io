# WEB前端设计项目资料
这个项目是西京学院电子商务专业《WEB前端设计》课程配套的网站资料，内含12个子项目，其中项目1为课程平台在线答题完成，项目2-12提交代码作品。

- 任务一：初识WEB前端设计
- 任务二：利用HTML标签创建网页元素
- 任务三：运用HTML布局样式进行网站布局
- 任务四：利用CSS样式美化网页
- 任务五：利用CSS+DIV进行页面布局
- 任务六：学习Javascript的基础知识
- 任务七：利用Javascript开发网页特效
- 任务八：学习jQuery的基础语法
- 任务九：利用jQuery创建事件  
- 任务十：利用jQuery制作动画
- 任务十一：利用Bootstrap对网页进行样式美化
- 任务十二：基于Bootstrap进行响应式网站开发
